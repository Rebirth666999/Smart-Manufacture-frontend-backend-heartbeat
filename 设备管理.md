## 3、主控节点

1、用户名：berry   密码：12345

2、主控节点控制长传送带、短传送带和轨道三个设备

3、启动主控节点：

`cd ~/HeartBeat/src/controller/controller`
`python3 controller.py`

4、如果要连后端，需要修改控制节点中的server_ip(在代码的头部)

5、项目路径为home下的Heartbeat

6、通过主控节点启动三个设备：

（1）、首先记住设备连接到树莓派USB口的顺序。

（2）、运行home目录下的 tty.sh，获取对USB的操作权限。

（3）、手段获取权限：

`sudo chmod 777 /dev/ttyUSB0`

`sudo chmod 777 /dev/ttyUSB1`

`sudo chmod 777 /dev/ttyUSB2`

（4）、查看树莓派连接USB的记录

`sudo dmesg | grep tty`

（5）、启动设备节点(x为数字，按照插入树莓的先后顺序依次为0、1、2)(顺序由步骤（1）确定)

`ros2 run track_server track_server /dev/ttyUSBx
ros2 run belt_short_server belt_short_server /dev/ttyUSBx
ros2 run belt_long_server belt_long_server /dev/ttyUSBx`

（6）、查看ROS2节点启动情况

`ros2 node list`

（7）、查看动作列表

`ros2 action list`

（8）、查看话题列表

`ros2 topic list`
